## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Worktree-leakage edits | Several executor agents misdirected Edit operations to `/mnt/data/src/nicu-assistant/...` (the main repo path) instead of their worktree path, causing commits to land on `main` directly. Worked around by careful merge inspection, but caused at least 2 dropped worktrees and 1 stash drama. | advisory | Future executors must verify `pwd` early and prefix every Edit `file_path` with the worktree-absolute path. The worktree-isolation contract isn't enforced by the Edit tool itself. |
| Em-dashes leak in non-comment user strings | DESIGN.md absolute ban on em-dashes was violated by Plan 5's drawer-summary work (`'—'` placeholder for null values). Critique re-run caught it. | advisory | Pre-commit grep for `—` in src/routes + src/lib (excluding `// comment` lines and `/* block comments */`) would catch this earlier. Plan 6 retroactively added the conformance pattern. |
| Nav clearance height assumed wrong constant | Initial `4rem` clearance assumed `spacing.16`, but actual mobile bottom-nav uses `min-h-14` (3.5rem). Compounded by missing the nav's `border-t` (1px). Took 3 commits to converge. | advisory | When sizing relative to another component, read the source rather than guess; spacing.14 vs spacing.16 + border-t = real gotcha. |
