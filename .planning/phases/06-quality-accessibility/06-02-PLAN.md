**Step 2: Create `e2e/morphine-wean-a11y.spec.ts`**

Import `{ test, expect }` from `@playwright/test` and `AxeBuilder` from `@axe-core/playwright`.

Write these tests:

1. **`test('morphine wean page has no axe violations in light mode')`**
   - Navigate to `/morphine-wean`
   - Wait for the page heading "Morphine Wean" to be visible (confirms page loaded)
   - Dismiss the disclaimer modal if it appears: check for a button containing "I understand" or "Acknowledge" text and click it. Use `page.getByRole('button', { name: /understand|acknowledge/i }).click({ timeout: 2000 }).catch(() => {})` to handle both cases.
   - Ensure `<html>` element has class `light` by running: `await page.evaluate(() => { document.documentElement.classList.remove('dark'); document.documentElement.classList.add('light'); })`
   - Run `const results = await new AxeBuilder({ page }).withTags(['wcag2a', 'wcag2aa']).analyze()`
   - Assert `expect(results.violations).toEqual([])`

2. **`test('morphine wean page has no axe violations in dark mode')`**
   - Same setup as light mode test
   - Switch to dark: `await page.evaluate(() => { document.documentElement.classList.remove('light'); document.documentElement.classList.add('dark'); })`
   - Run axe with same tags, assert zero violations

3. **`test('morphine wean page has no axe violations with schedule visible')`**
   - Navigate to `/morphine-wean`, dismiss disclaimer if present
   - Fill in valid inputs to trigger schedule rendering:
     - Fill "Dosing weight" input with "3.1"
     - Fill "Max morphine dose" input with "0.04"
     - Fill "Decrease per step" input with "10"
   - Wait for "Step 1" text to be visible (schedule rendered)
   - Run axe with wcag2a + wcag2aa tags, assert zero violations

For filling inputs, use `page.getByLabel('Dosing weight')` etc. If labels don't match exactly, use `page.locator('#morphine-weight')` as fallback (IDs are set in the component).

Add a `test.describe('Morphine Wean Accessibility')` wrapper around all tests.

Add a shared `test.beforeEach` that navigates to `/morphine-wean` and dismisses the disclaimer modal.
