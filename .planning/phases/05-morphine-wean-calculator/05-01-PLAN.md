    - Compounding mode: weight=3.1, maxDose=0.04, decreasePct=0.10 => step1 dose = 0.124mg, step2 dose = 0.1116mg, step3 dose = 0.10044mg, reduction decreases each step
    - Both modes: step 1 always has dose = weight * maxDose, reduction = 0 (first step, no reduction from previous)
    - Linear: all reduction amounts after step 1 are equal (weight * maxDose * decreasePct)
    - Compounding: each reduction = previousDose * decreasePct
    - Always exactly 10 steps returned
    - Edge case: if dose would go negative in linear mode, clamp to 0

    **Create `src/lib/morphine/types.ts`:**
    ```typescript
    export type WeanMode = 'linear' | 'compounding';

    export interface WeanStep {
      step: number;           // 1-based step number
      doseMg: number;         // Dose in mg
      doseMgKgDose: number;   // Dose in mg/kg/dose
      reductionMg: number;    // Reduction from previous step in mg (0 for step 1)
    }

    export interface MorphineStateData {
      activeMode: WeanMode;
      weightKg: number | null;
      maxDoseMgKgDose: number | null;
      decreasePct: number | null;
    }
    ```

    **Create `src/lib/morphine/calculations.ts`:**
    Export two functions:

    `calculateLinearSchedule(weightKg: number, maxDoseMgKgDose: number, decreasePct: number): WeanStep[]`
    - Step 1: doseMg = weightKg * maxDoseMgKgDose, reductionMg = 0
    - Steps 2-10: reductionMg = weightKg * maxDoseMgKgDose * decreasePct (constant), doseMg = previousDoseMg - reductionMg, clamp doseMg to 0 if negative
    - doseMgKgDose = doseMg / weightKg for each step

    `calculateCompoundingSchedule(weightKg: number, maxDoseMgKgDose: number, decreasePct: number): WeanStep[]`
    - Step 1: doseMg = weightKg * maxDoseMgKgDose, reductionMg = 0
    - Steps 2-10: doseMg = previousDoseMg * (1 - decreasePct), reductionMg = previousDoseMg - doseMg
    - doseMgKgDose = doseMg / weightKg for each step

    Both always return exactly 10 steps (per D-04).
