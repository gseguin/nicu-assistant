- **v1.6 NumericInput `showRangeHint` → v1.7 `showRangeError`:** Symmetric opt-out prop pattern. Both props default to `true` (non-breaking) and are applied together on Starting Volume.
- **v1.3 packets auto-reset → v1.7 packets auto-select:** Both behaviors now live in a single `$effect` with two branches, both gated on `formulaSupportsPackets(id)` (data-driven helper).
- **v1.5 searchable SelectPicker → v1.7 row layout:** Formula picker's search dialog still opens from inside the flex row. `min-w-0` handling on the flex child prevents overflow bugs.
- **v1.6 SegmentedToggle → v1.7 row layout:** Base toggle still sits above the row on its own line (intentional — Base is a primary decision that deserves its own row).
