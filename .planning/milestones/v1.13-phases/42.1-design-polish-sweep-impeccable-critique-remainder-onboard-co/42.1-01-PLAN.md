# Plan 42.1-01: Chrome Alignment + Mobile-Nav Clearance

## Objective

Enforce DESIGN.md's Identity-Inside Rule on global chrome (top desktop tab, mobile bottom tab, hamburger drawer star icons), shorten the morphine tab label, fix universal `pb-20` mobile-nav clipping with safe-area math, add the sole-licensed glassmorphism on the bottom nav surface, and add a static conformance test that prevents identity-color regression in chrome. Ships in Wave 1 because it's independently shippable, touches the highest-traffic surfaces (every route's chrome), and unblocks user-visible value (no more occluded content, no more 5-hue chrome bleed) without depending on `<HeroResult>`.
